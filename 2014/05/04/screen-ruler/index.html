Simple Demo of Screen Ruler in Web Design

2014-05-04

今天看搜到了這個有用的工具 - Screen Ruler，很簡單的一行 CSS Code，就可以在瀏覽器的邊緣刻劃出尺規，對於設計網頁版面很外行的我而言算是個福音！

不過實際做了一個 div 來測了一下，發現位移會有點誤差，大約在 7 pixel 左右，也許是我的螢幕問題或是網頁版面設計本身的障礙，但是長度上是精準的
