[JavaScript] TaffyDB

2014-05-29

TaffyDB 是一個輕量級及 Open Source 的 JavaScript Database

它的專案內容很小，主要的 library 只有一個 taffy.js，所以它的操作應該都是在瀏覽器執行的程式記憶體中，沒有寫入到檔案系統

Taffy 是 ORM 實作的一種，它也有提供 extension 讓網頁程式可以介接 MySQL 等資料庫，像 mongoDB 那樣
